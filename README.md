## Convention

### Commit Message

| 태그     | 설명                                             |
| -------- | ------------------------------------------------ |
| feat     | 새로운 기능 추가                                 |
| fix      | 버그 수정                                        |
| refactor | 코드 리팩토링 (기능 변화 없이 코드 구조 개선)    |
| chore    | 빌드 프로세스나 보조 도구 수정 (코드 변경 없음)  |
| docs     | 문서 수정                                        |
| test     | 테스트 코드 추가/수정                            |
| style    | 코드 스타일 수정 (포매팅, 세미콜론 추가/제거 등) |
| perf     | 성능 개선                                        |
| ci       | CI 설정 파일 변경 (GitHub Actions, Jenkins 등)   |
| revert   | 이전 커밋 되돌리기                               |
